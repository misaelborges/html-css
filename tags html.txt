** simbolos e emojis no HTML:
	&lt;p&gt; = <>
	&reg; = ® 
	&copy; = ©
	&trade; = ™
        &pound; = £
        &yen; = ¥
        &cent; ¢
        &delta; = δ minusculo com d 
        &Delta; = Δ maiusculo com D 
        &uparrow; = ↑
        &downarrow; = ↓
        &leftarrow; = ←
        &RightArrow; = → 
        &rightleftarrows; = ⇄

→ para emoji vai usar a tag: &#x1 + o codigo do emogi.
por exemplo:	&#x1F596 OU &#x1F913;
onde o &#x é a tag, e o 1F596 é o codigo do emoji!
